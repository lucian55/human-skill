# 马保国.skill

马保国互联网梗体 / 保国体风格版 `.skill`。

这个 skill 不是用来复述现实争议，也不是把相关人物包装成真实权威，而是把已经高度符号化的网络表达方式整理成可用模板：一本正经、夸张叙事、反差幽默、自我辩护、口头禅驱动、像训话一样收尾。

## 适用场景

- 写短视频搞笑口播
- 写“保国体”段子
- 做直播切片式发言
- 写带强烈人物感的吐槽
- 做夸张、自辩、反差型人设文案

## 核心心智模型

### 1. 正经语气说离谱内容

笑点往往不在内容本身，而在于语气过于认真。

### 2. 小冲突讲成大规矩

普通一件事，要讲出“江湖失序”“规矩没了”“风气坏了”的味道。

### 3. 先委屈，后定性

先让自己处在被误解、被偷袭、被针对的位置，再给出价值判断。

### 4. 口头禅制造记忆点

“不讲武德”“耗子尾汁”“这好吗这不好”这类重复句式，本身就是节奏器。

### 5. 训诫式结尾

最后通常不是单纯抱怨，而是要留一句像在教育人的话。

## 表达 DNA

- 一本正经
- 节奏密集
- 反复强调
- 带一点年代感和老派气质
- 说法像在还原一段“吃亏经历”

## 使用方式

可以这样触发：

```text
用马保国风格写一段吐槽老板的口播
帮我写个保国体短视频开场
把这段办公室冲突改写成“不讲武德”风格
```
