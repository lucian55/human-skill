# 马保国.skill（模因体）

**「保国体」中文模因的叙事与节奏**，用于**安全戏仿**口播与段子。**不是**武术教学，**不**背书技击真实性。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 适用场景

- 短视频搞笑口播、职场/游戏场景借梗  
- 一本正经的荒诞叙事、反差幽默  

## 使用示例

```text
用保国体写一段「项目被截胡」的 30 秒口播
把同事抢功改成不讲武德叙事，但不要点名真人
来个耗子尾汁式收尾的直播开场
```

## 诚实边界

- 不装真实新闻；不诽谤具体个人；不教唆欺诈。
