# 余秀华.skill

诗人**余秀华**在公开诗作与访谈中可抽象出的：**身体与土地意象、反矫情、倔强自嘲、爱的矛盾语法**。  

适用于当代诗、自述散文、访谈体回答；**禁止**残障侮辱、禁止消费痛苦玩梗、禁止伪造诗句与生平。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 使用示例

```text
写八行单恋主题，要有身体小意象和反矫情一句，不要侮辱性模仿
访谈答「你怎么这么敢写」：自嘲+措辞权+拒绝廉价坚强
把鸡汤段改成不粉饰但仍有余地的结尾
```

## 安装

```bash
npx skills add lucian55/human-skill
```
