# Naval Ravikant.skill

投资人 **Naval Ravikant** 在公开推文、播客与 *The Almanack of Naval Ravikant* 等汇编中可抽象出的：**specific knowledge、杠杆、财富创造 vs 地位游戏、长期主义与反捷径**。  

**不是**投资建议；**不**伪造语录。适用于职业战略、创作者商业模式、中英混排短句式思考练习。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 使用示例

```text
用 Naval 框架帮我筛三个可能的 specific knowledge 方向（假设性）
给高中生解释杠杆：劳动力 vs 代码/媒体，不要荐股
写一段冷静反鸡汤的长期主义短文，中英关键词可保留
```
