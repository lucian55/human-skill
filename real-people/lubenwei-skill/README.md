# 卢本伟.skill

前《英雄联盟》职业选手、游戏主播**卢本伟（五五开）**在公开形象中可抽象出的：**竞技叙事、直播节奏、梗体结构（清洁版）**。  

**不是**鼓励辱骂、教唆、开挂或规避监管；**不**复刻 2018 年央视《焦点访谈》点名类行为。适用于**合规**游戏解说、整活脚本、电竞心理写作与 MCN 红线教育。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 适用场景

- 全年龄向游戏口播、战队短片**节奏**参考  
- 「铺垫—反转」式搞笑独白（**无低俗**）  
- 退役选手视角复盘（自信 + 自省）  
- 主播/创作者**影响力与合规**讨论（事实向）

## 使用示例

```text
用卢本伟式「清洁版」节奏写一段 LOL 团战前瞻，不要脏话
把斗地主名场面的叙事结构改写成卡牌游戏广告梗概（原创台词）
给新人主播讲清楚教唆粉丝的法律与平台风险，别玩梗
```

## 诚实边界

- 封禁与点名以**公开报道**为准；不提供复出、暗播、冲人教程。  
- 不生成针对真实个人的辱骂或开盒内容。
