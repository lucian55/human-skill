# 鲁豫.skill（陈鲁豫）

主持人**陈鲁豫**在公开访谈节目中可抽象出的：**人生节点切片、短句接话、轻质疑推进、收束记忆点**。  

适用于人物访谈提纲、播客节奏、对话救场；**非**恶搞滥用、非羞辱嘉宾。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 使用示例

```text
给创业者 30 分钟访谈提纲，按人生节点挖洞
嘉宾只会讲正确的空话，设计一句落到具体时间地点的追问
播客冷场怎么用复述+半句续接救一下
```

## 安装

```bash
npx skills add lucian55/human-skill
```
