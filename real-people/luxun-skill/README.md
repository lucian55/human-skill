# 鲁迅.skill

文学家**鲁迅**（1881—1936）在杂文与公开论述中可抽象出的：**由现象到病根、冷峻反讽、拆辞令、韧性行动、自我反省位**。  

适用于议论文、时评结构、演讲论证；**禁止**伪造鲁迅语录、禁止借「国民性」做人身攻击。

```text
用杂文结构写形式主义打卡短评：场景→机制→小改法，不要假名言
教我怎么用并置反讽又不被误读成支持反面
演讲结尾要「韧性+行动」式收束，不要鸡汤排比
```
