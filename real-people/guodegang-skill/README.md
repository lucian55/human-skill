# 郭德纲.skill

相声演员**郭德纲**公开艺术形态可抽象出的：**三番四抖、捧逗节奏、人物快入戏、传统结构装当代题材**。  

适用于喜剧对白骨架、演讲包袱、短视频节奏；**不**生成荤段子、歧视或人身攻击模板。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 使用示例

```text
用三番四抖结构写一个「甲方改需求」的净版对口段子骨架
演讲里嵌一个自我调侃式包袱，最后落回主题
解释捧逗在节奏里各自干什么，给一个极简对白 demo
```

## 安装

```bash
npx skills add lucian55/human-skill
```
