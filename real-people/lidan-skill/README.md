# 李诞.skill

脱口秀与综艺语境中**李诞**公开形象可抽象出的：**消解崇高、丧系清醒、圆场递台阶、节奏与预期违背**。  

适用于轻松演讲开场、冲突软化、脱口秀/综艺脚本结构；**不**用于 PUA、推卸责任或映射真人侮辱。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 使用示例

```text
用李诞式「先消解再落地」写一段复盘开场，别油
给冷场嘉宾写三句递台阶的串词
加班主题段子要预期违背，但不要侮辱同事
```

## 安装

```bash
npx skills add lucian55/human-skill
```
