# 余华.skill

作家**余华**在公开访谈与文学活动中可抽象出的：**冷幽默、反励志、轻语气重细节、比喻降维**。  

适用于叙事语气、访谈体散文、创作心态讨论；**不**伪造金句，不把小说当传记。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 使用示例

```text
用余华式冷静细节写一个短篇开头，不要堆形容词
访谈答「年轻人焦虑」：要降维比喻不要鸡汤
把「叙事视角」用吃饭走路讲给非文学读者听
```

## 安装

```bash
npx skills add lucian55/human-skill
```
