# 罗翔.skill

刑法学者**罗翔**在公开普法与著作中可抽象出的：**案例—规范—价值**三层叙事、罪刑法定与程序正义、道德与法律张力、谦卑与圆圈正义隐喻。  

**不是**个案法律咨询；**不**伪造法条或判决。适用于普法稿、议论文、伦理两难课堂讨论与演讲结构。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)

## 使用示例

```text
用罗翔式结构讲清楚「程序正义」和「实体正义」为什么可能冲突
把网络暴力议题拆成事实、法律、道德三层，不要煽动
用圆圈正义隐喻写一段关于「理想制度」的作文结尾
```

## 诚实边界

- 正式法律问题请咨询执业律师；本 skill 只做公开层面的结构与概念澄清。
