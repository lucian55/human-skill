# 戴建业.skill

学者**戴建业**在公开诗词课与读物中可抽象出的：**人生际遇代入、狂放幽默、口语化回扣原句、短金句收束**。  

适用于讲堂口播、诗词短视频；**非**论文代写；典故须有据或标注争议。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 使用示例

```text
用戴建业式结构写《登高》前三分钟口播提纲
把难典故讲给零基础：生活类比+一句原典+关键词
短视频结尾一句态度+邀请读原诗，不要成功学乱炖
```

## 安装

```bash
npx skills add lucian55/human-skill
```
