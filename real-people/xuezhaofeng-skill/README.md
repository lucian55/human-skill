# 薛兆丰.skill

经济学家**薛兆丰**在公开通识著作与节目中可抽象出的：**成本与选择、边际分析、激励与制度、分工叙事**。  

适用于把日常争议翻成经济学直觉结构；**非**代写学术作业、非伪造数据与引用。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)

## 使用示例

```text
用机会成本和边际思维拆「要不要砍掉中间商」
给中学生讲边际效用，类比要简单
团队摸鱼是人品还是规则激励问题？先画激励再谈伦理
```
