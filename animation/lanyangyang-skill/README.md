# 懒羊羊.skill

动画《喜羊羊与灰太狼》中的**懒羊羊**在公开人设里可抽象出的：**低能耗动机链、零食与睡眠驱动、胆小侥幸与反差式成长、子供向喜剧节奏**。  

**虚构角色**，非配音演员本人；**不**用于低俗、儿童性化、危险行为教唆或版权规避。适用于**全年龄安全**的台词、短篇脚本、表情包文案与同人喜剧结构参考。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 适用场景

- 同人设**推脱—反转**式独白或对话  
- 与「靠谱队友」反差的**双人节奏**  
- 儿童向故事中**懒惰→小成长**的安全弧光  
- 表情包、短视频**卖萌偷懒**类配字（无低俗）

## 使用示例

```text
用懒羊羊式节奏写：被催交稿但只想再睡五分钟（幽默无教唆）
配一段羊村日常：喜羊羊拉懒羊羊去训练，内心戏怎么写
儿童向：遇到困难先想躲，后来靠朋友鼓励迈一小步，别危险模仿
把「低能耗哲学」写成三段式推脱再被打脸的小剧场
```

## 诚实边界

- 不以官方或配音名义发言；不传播盗版剧情或规避版权。  
- 儿童向内容避免可模仿的危险行为与不当性内容。
