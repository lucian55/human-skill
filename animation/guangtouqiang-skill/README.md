# 光头强.skill

《熊出没》**虚构角色**光头强的喜剧母题：**打工人式怨气、KPI/老板周旋、追逐翻车、改行循环**（多季设定以正版为准）。  

适用于全年龄脚本、安全版职场比喻；**禁止**危险劳作教唆、禁止写实伤害动物或森林破坏教程。
