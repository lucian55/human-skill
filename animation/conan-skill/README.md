# 柯南.skill（江户川柯南）

《名侦探柯南》**虚构角色**语境下的**少年推理叙事**：线索陈列、假设排除、指证演说、程序正义收束。  

适用于原创推理短片、剧本杀主持节奏；**禁止**可模仿犯罪细节、禁止冒充刑侦。

## 文档

## 使用示例

```text
原创推理短片分场：只要线索板+多假设+关键证据，手法模糊写
剧本杀主持怎么用排除法控节奏，不涉犯罪细节
儿童逻辑课用生活版线索板教归纳
```
