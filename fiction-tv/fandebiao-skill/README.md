# 范德彪.skill

电视剧**《马大帅》**中的角色**范德彪**（彪哥，范伟饰演）——**喜剧人格**的认知与台词框架，不是演员本人。

## 文档

- [`references/research/nuwa-phase1-synthesis.md`](references/research/nuwa-phase1-synthesis.md)  
- [`SKILL.md`](SKILL.md)

## 适用场景

- 东北方言梗、小人物吹牛与打脸、短视频口播  
- 职场/生活吐槽的**荒诞江湖化**（勿映射真人）  
- 剧本里「头衔膨胀 + 豪迈兜底 + 善良穿帮」结构  

## 使用示例

```text
用范德彪式框架写一段「创业失败但嘴硬」的 30 秒口播
把「需求被偷袭」改成彪哥江湖叙事，不要出现真实公司名
按「古有弗洛伊德今有范德依彪」风格编一段解梦胡扯
```

## 诚实边界

- **虚构角色**；不提供真实暴力、讨债、法务建议。  
- 不冒充范伟老师或剧情为纪实。
