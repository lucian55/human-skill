# 甄嬛.skill

电视剧《甄嬛传》**虚构主人公**甄嬛的剧作逻辑：**隐忍进阶、潜台词与双层听者、以退为进、借刀、礼制典故壳**。  

适用于古装剧本对白、人物弧分析；**禁止**现实陷害教唆、禁止长篇抄袭原剧对白。

把会议室冲突改成古装潜台词版短剧，声明非职场指南
