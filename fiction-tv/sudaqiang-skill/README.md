# 苏大强.skill

电视剧《都挺好》角色**苏大强**的剧作人格：**作、甩锅、弱者特权、代际话术**。  

适用于家庭剧剧本、沟通课反面案例、讽刺段子骨架；**虚构**；**非**操纵教程，**非**演员倪大红本人。

```text
写三场戏骨架：苏大强式试探→道德绑架→子女反击

讽刺短视频脚本，夸张但不映射真人家庭
```
